Anthropic just made its most significant enterprise move yet. Claude Cowork’s February 2026 update does not just add features; it repositions Claude as a configurable infrastructure layer that enterprises can shape to their exact operational workflows.

This is what separates it from generic AI assistants. The update gives IT admins, department heads, and knowledge workers a unified system to build, manage, and deploy AI agents that follow how their organization actually works, not a one-size-fits-all chatbot.

Here is a complete breakdown of every confirmed change in this release and what it means for real teams.

## **Private Plugin Marketplaces: Enterprise-Grade Control Arrives**

The most consequential shift in this release is the introduction of private plugin marketplaces. Admins can now build custom plugins from starter templates or from scratch, with Claude guiding setup by asking questions to tailor skills, commands, and connectors (MCPs) to the company’s specific needs.

All plugin management now lives inside a unified menu called “Customize,” consolidating plugins, skills, and connectors into a single dashboard. This eliminates the scattered admin experience that enterprise IT teams previously navigated across multiple settings panels.

Per-user provisioning and auto-install capabilities mean admins can push the right tools to the right teams without manual onboarding. Private GitHub repositories as plugin sources are available in private beta, giving engineering teams the ability to version-control and audit every plugin deployed organization-wide.

Anthropic designed each new role-specific plugin with practitioners in the relevant field, so terminology, workflows, and outputs reflect how work actually gets done.

- **Investment Banking:** Review transaction documents, build comparable company analyses, and prepare pitch materials
- **Equity Research:** Parse earnings transcripts, update financial models with new guidance, and draft research notes
- **Private Equity:** Review large document sets, extract standardized financial data, model scenarios, and score investment opportunities against criteria
- **Wealth Management:** Analyze portfolios, identify drift and tax exposure, and generate rebalancing recommendations at scale

## **Frequently Asked Questions (FAQs)**

### What is Claude Cowork and how does it differ from standard Claude?

Claude Cowork is Anthropic’s enterprise workspace that layers role-specific plugins, connectors, and admin controls on top of Claude. Unlike the standard chat interface, Cowork lets organizations configure Claude with custom workflows, private marketplaces, and department-specific agents built around their own terminology and processes.

### Who can access the new enterprise plugin features in February 2026?

All plugin user experience updates are available to all Cowork users. Company branding, per-user provisioning, and MCP connector controls require a Team or Enterprise plan. Cross-app Excel and PowerPoint integration is available as an early research preview for all paid plans on Mac and Windows.

### What new connectors were added in the February 2026 update?

New connectors include Google Workspace (Calendar, Drive, Gmail), Docusign, Apollo, Clay, Outreach, Similarweb, MSCI, LegalZoom, FactSet, WordPress, and Harvey. Companies including Slack by Salesforce, LSEG, S&P Global, Apollo, Common Room, and Tribe AI have also built plugins for joint enterprise customers.

### How does the Excel to PowerPoint cross-app feature work?

Claude runs a data analysis inside Excel, then uses the context from that add-in to build a presentation in PowerPoint. The full workflow runs as a single assigned project. Users need to download both the Claude add-in for Excel and the Claude add-in for PowerPoint separately to enable this capability.

### Can organizations build their own plugins for Claude Cowork?

Yes. Admins can build plugins from starter templates or from scratch, with Claude guiding setup interactively. Plugins are file-based and available on GitHub. Private GitHub repositories as plugin sources are currently in private beta. Organizations can also set up private marketplaces to distribute custom plugins across teams.

### What is OpenTelemetry support in Claude Cowork and why does it matter?

OpenTelemetry is an open-source observability standard. Anthropic’s integration lets admins track plugin usage, AI tool activity, and cost data across the organization in a format compatible with existing monitoring infrastructure. This addresses a core requirement for enterprise IT and compliance teams evaluating AI deployment at scale.

### What does the Brand Voice plugin by Tribe AI do?

The Brand Voice plugin analyzes an organization’s existing documents, marketing materials, and conversations to extract voice patterns and distill them into clear, enforceable brand guidelines. It gives content and marketing teams consistent AI output that matches established brand standards.
